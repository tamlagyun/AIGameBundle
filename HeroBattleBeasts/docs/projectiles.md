# 子弹飞行与命中判定说明

## 1. 目标

第 14 步把子弹飞行、生命周期和敌人命中判定放入核心玩法层。预览和 Cocos 接入层只展示结果，不再自行判断子弹是否命中敌人。

## 2. 当前行为

- `fireWeapon` 创建玩家子弹。
- 子弹继承玩家朝向，按武器配置中的 `bulletSpeed` 水平移动。
- 子弹记录 `ageSeconds`。
- `updateProjectiles` 每帧推进子弹位置。
- 子弹达到 `lifetimeSeconds` 后会被移除。
- 玩家子弹与未击败敌人的碰撞区域相交时，会造成伤害并移除子弹。
- 敌人生命归零后会增加击败数量和分数，并继续触发通关目标评估。

## 3. 配置字段

关卡配置中的 `combat` 字段当前包含：

- `bulletBounds`：子弹命中判定尺寸。
- `enemyBounds`：敌人受击判定尺寸。
- `pickupBounds`：道具自动拾取判定尺寸。
- `bulletLifetimeSeconds`：子弹存活时间。

## 4. 边界

当前暂不包含：

- 敌人子弹。
- 多段命中。
- 子弹打墙或打平台。
- 暴击、元素、击退等复杂战斗效果。
- Cocos 物理碰撞组件接入。

这些内容后续步骤再单独确认并实现。
