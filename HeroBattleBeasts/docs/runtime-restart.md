# 运行时重开说明

## 1. 目标

第 17 步为运行时增加关卡重开能力。胜利或失败后，不需要刷新页面或重建外部 Cocos 场景，只需要调用运行时重启入口即可恢复当前关卡初始状态。

## 2. 当前行为

- `GameRuntime` 构造时会保存标准化后的配置。
- `restart` 会用保存的配置重新创建初始 `GameState`。
- 重启后状态恢复为 `playing`。
- 时间、金币、击败数、子弹、敌人、道具和玩家状态都会恢复初始值。
- 平台、物理、战斗和关卡目标配置会保留。
- Web 预览的 `R` 键现在调用 `runtime.restart()`，不再刷新页面。

## 3. 后续接入

Cocos UI 的重开按钮应直接调用 `GameRuntime.restart()`，再用返回的 `RuntimeViewModel` 刷新节点表现。

## 4. 边界

当前只支持重开当前关卡，不包含：

- 切换关卡。
- 存档回滚。
- 复活消耗。
- 结算奖励确认。

这些内容后续步骤再单独确认并实现。
