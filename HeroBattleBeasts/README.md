# HeroBattleBeasts

`HeroBattleBeasts` 是一个计划中的卡通 2D 横版动作小游戏。

项目目标是把魂斗罗式跑跳射击与冒险岛式平台探索结合起来，先做一个 Web 可运行原型，再逐步扩展成更完整的小游戏。

## 当前状态

已完成第 1 步初始化：

- 创建独立项目目录 `HeroBattleBeasts`
- 写入项目协作规则 `AGENTS.md`
- 记录项目目标、风格和分步骤开发流程

当前还没有开始实现玩法代码。

## 开发规则

本项目按步骤推进：

1. 每一步先列计划。
2. 等用户确认。
3. 再实施该步骤。
4. 实施后说明完成内容和验证结果。

## 初始游戏方向

- 风格：卡通、明亮、横版 2D。
- 核心体验：移动、跳跃、射击、击败怪物、收集奖励、推进关卡。
- 魂斗罗参考点：快节奏射击、敌人波次、武器强化、闪避弹幕。
- 冒险岛参考点：平台地形、怪物、金币道具、轻探索和角色成长感。
